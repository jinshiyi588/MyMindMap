MindMap主要介绍和设计思路

一：简介：
MindMap
1：是结合现在流行的记忆方式：画脑图，来方便地合理地设计用户自己的脑图的管理型学习软件；
2：主要功能是通过拖动提供的图形放置在画布的合理位子构建记忆框架，并且可以在每个图形内添加帮助自己记忆的关键词或图片；
（关键词有字数限制，图片每个位置只能放置一张）
3：脑图画好后，有提示用户按时复习的功能（根据艾宾浩斯记忆曲线设计），并将提醒设置到手机的状态栏；

二：设计思路：
MindMap
1：可以模仿画图程序，要有新建，保存，打开，浏览的基本功能；
2：要有数据库，但是数据库只是存储已有脑图名和复习时间，方便查看时间和提醒；
